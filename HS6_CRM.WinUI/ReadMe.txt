UI => Kullanıcının göreceği ekranların bulundupu katman
BL(Businnes Layer) => Kullanıcının iş analitiğnin yapıldığı katman
DAL => Database erişim katmanı
Entities => Database tarafında bulunan tablolardaki nesnelerin bulunduğu katman
DTO (Data transfer object) => Entities katmanı UI dan gelen verinin map edildiğ katman
CORE => Diğer katmanların ihtiyaç duyduğu nesneleri işlmelerin bulunduğu katman
